# TrabalhoAgentes-Thiago-Oliveira-Luiz-Santos-e-Vitor-Lucas

O agente consiste em um robô inserido em um ambiente representado por um grid 5x5. Ele inicia em uma posição inicial e deve se deslocar pelo ambiente utilizando movimentos nas quatro direções (cima, baixo, esquerda e direita) até alcançar uma posição objetivo previamente definida. O robô toma decisões com base em sua posição atual, respeitando os limites do ambiente.
